# Soluções Beecrowd

Bem-vindo ao meu repositório de soluções para os desafios do [beecrowd](https://www.beecrowd.com.br/).

Este repositório reúne códigos desenvolvidos em várias linguagens, com ênfase na melhoria da lógica de programação. Em especial, os exercícios em C foram amplamente utilizados para reforçar o aprendizado na disciplina de Algoritmos e Lógica de Programação do **CEFET-MG**.
