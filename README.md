## 🛠️ Tecnologias e Linguagens

Os desafios são resolvidos focando em eficiência e boas práticas. Os códigos em **C** foram desenvolvidos especialmente para a disciplina de *Algoritmos e Lógica de Programação* no **CEFET-MG**.

## 📌 Sobre o Repositório

O objetivo central deste repositório é servir como um arquivo de estudos e não apenas "resolver o problemas". 

> [!NOTA]  
> O foco principal é o desenvolvimento do raciocínio lógico e a transição suave entre diferentes sintaxes de linguagens.

---

## 📂 Organização

Para facilitar a navegação, as soluções estão organizadas por linguagem ou categoria do Beecrowd:

| Pasta | Descrição |
| :--- | :--- |
| `/C` | Foco em fundamentos e performance (**CEFET-MG**). |
| `/Csharp` | Familiarização e Prática Acadêmica (**PUC Minas**) |
| `/Python` | Familiarização com a sintaxe e as bibliotecas padrão da linguagem |
| `/Java` | Familiarização com a sintaxe e as bibliotecas padrão da linguagem |
